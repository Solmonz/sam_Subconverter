```shell

```

方法二：
替换订阅链接到下面的链接中

```shell
https://sub.1688.al/sub?target=clash&insert=true&new_name=true&scv=true&udp=true&exclude=&include=&url=订阅地址&config= https://ghproxy.com/https://raw.githubusercontent.com/Solmonz/sam_Subconverter/main/Sub.ini
```  

方法三：
订阅转换站

```shell

```
